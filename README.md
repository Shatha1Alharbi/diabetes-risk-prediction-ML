# Healthcare Data Mining & Machine Learning Project

## Overview
This project focuses on healthcare data mining and machine learning techniques for disease risk prediction and medical data analysis. The project includes literature analysis of recent healthcare machine learning studies, predictive model implementation, and model evaluation using multiple classification algorithms.

The project was developed as part of the Data Warehousing and Data Mining course at Princess Nourah University.

---

## Project Objectives
- Explore the role of machine learning in healthcare prediction systems
- Analyze and compare machine learning algorithms used in healthcare studies
- Implement predictive classification models using healthcare datasets
- Evaluate model performance using standard machine learning metrics
